# Data

- **mortality.csv** — Maternal health risk dataset (1014 rows). Features: Age, SystolicBP, DiastolicBP, BS (blood sugar in mmol/L), BodyTemp, HeartRate, RiskLevel (low/mid/high).
- **main.csv** — Pima Indians Diabetes dataset (768 rows). Features: Pregnancies, Glucose (mg/dL), BloodPressure, BMI, Age, Outcome (0/1).
- **preg.xlsx** — Raw hospital ANCC register. Not yet cleaned for ML use.
